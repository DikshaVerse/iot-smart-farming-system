# IoT Smart Farming System

An IoT-based smart farming monitoring system designed to help farmers monitor environmental conditions in real time using sensors and cloud technology.

The system collects sensor data such as soil moisture, temperature, and motion detection and sends it to the ThingSpeak cloud platform for monitoring and analysis.

---

## Features

- Soil moisture monitoring
- Temperature sensing
- Motion detection for animal intrusion
- Cloud monitoring using ThingSpeak
- Real-time environmental data tracking

---

## Hardware Components

- Arduino UNO
- Soil Moisture Sensor
- Temperature Sensor
- Motion Sensor
- ESP Module (WiFi communication)

---

## System Architecture

Sensors → Arduino → ESP Module → ThingSpeak Cloud → Dashboard

---

## Applications

- Smart agriculture
- Remote farm monitoring
- Environmental condition tracking
- Agricultural data analysis
